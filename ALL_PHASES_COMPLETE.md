## ğŸš€ Quick Start Guide

```bash
# Install
pip install --upgrade tars-cli

# Verify
tars setup

# Configure
tars config set default.namespace production
tars config set default.cpu 90
tars config set default.webhook_url https://hooks.slack.com/...

# Use Features
```
